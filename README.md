## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/mlanies/bussy-web.git` 

2.  Перейдите в директорию проекта:
    
    `cd bussy-web` 
    
3.  Установите зависимости:
    
    `npm install` 
    

## Использование

Опишите, как использовать ваш проект. Это может включать команды для запуска приложений, настройки окружения и так далее.

`npm start` 

Теперь проект доступен по адресу  `http://localhost:3000`.

## Вносить изменения

Если кто-то желает внести изменения или улучшения, они могут:

1.  Форкнуть проект.
2.  Создать новую ветку:  `git checkout -b feature/ИмяВетки`.
3.  Внести изменения и закоммитить их:  `git commit -m 'Описание изменений'`.
4.  Отправить изменения в ваш форк:  `git push origin feature/ИмяВетки`.
5.  Создать Pull Request.
